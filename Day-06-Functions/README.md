# 📖 What are Functions?

Functions are reusable blocks of code that perform a specific task.

Benefits:
- Reduce code repetition
- Improve readability
- Make programs modular

---

# 📖 Creating a Function

Syntax:

```python
def function_name():
    print("Hello Python")
```

Calling a Function:

```python
function_name()
```

---

# 📖 Function with Parameters

Parameters allow data to be passed into a function.

Example:

```python
def greet(name):
    print("Hello", name)

greet("Sam")
```

Output:

```text
Hello Sam
```

---

# 📖 Return Statement

The `return` keyword sends a value back from a function.

Example:

```python
def add(a, b):
    return a + b

result = add(5, 3)
print(result)
```

Output:

```text
8
```

---

# 📚 Key Points

- Functions help organize code.
- Use `def` to define a function.
- Parameters allow input to functions.
- `return` sends a value back.
