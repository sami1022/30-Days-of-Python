# 📖 What are Variables?

Variables are containers used to store data values.

Example:
```python
name = "Sam"
age = 20
```

---

# 📖 Data Types in Python

| Data Type | Example |
|-----------|---------|
| String | "Hello" |
| Integer | 10 |
| Float | 5.7 |
| Boolean | True |

---

# 💻 Example Program

```python
name = "Sam"
age = 20
height = 5.7
is_student = True

print("Name:", name)
print("Age:", age)
print("Height:", height)
print("Student:", is_student)
```

---

# 📚 Key Points

- Variables store data.
- Python supports multiple data types.
- `print()` is used to display output.
- `type()` is used to check data type.
