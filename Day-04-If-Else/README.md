# 📖 What is If Statement?

The `if` statement is used to check conditions.

Example:
```python
age = 18

if age >= 18:
    print("Eligible to vote")
```

---

# 📖 If-Else Statement

Used when there are two possible conditions.

Example:
```python
num = 10

if num % 2 == 0:
    print("Even Number")
else:
    print("Odd Number")
```

---

# 📖 If-Elif-Else Ladder

Used to check multiple conditions.

Example:
```python
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 60:
    print("Grade B")
else:
    print("Grade C")
```

---

# 💻 Example Program

```python
age = int(input("Enter your age: "))

if age >= 18:
    print("You can vote")
else:
    print("You cannot vote")
```

---

# 📚 Key Points

- `if` checks a condition.
- `else` runs when condition is false.
- `elif` is used for multiple conditions.
- Indentation is important in Python.
