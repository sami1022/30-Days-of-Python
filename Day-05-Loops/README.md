# 📖 What are Loops?

Loops are used to execute a block of code multiple times.

---

# 📖 for Loop

The `for` loop is used to iterate over a sequence.

Example:
```python
for i in range(5):
    print(i)
```

Output:
```text
0
1
2
3
4
```

---

# 📖 while Loop

The `while` loop runs until the condition becomes false.

Example:
```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

---

# 📖 break Statement

`break` is used to stop the loop.

Example:
```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

---

# 📖 continue Statement

`continue` skips the current iteration.

Example:
```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

---

# 📚 Key Points

- Loops reduce repetitive code.
- `for` loop is commonly used with `range()`.
- `while` loop depends on conditions.
- `break` stops the loop.
- `continue` skips one iteration.
